# Company Structure

REST API for managing company structure.

### Run

Clone the repository:

```bash
git clone https://github.com/kocifajadrian/CompanyStructure.git
cd CompanyStructure
```

Set your database connection string in `src/CompanyStructure.Api/appsettings.Development.json`:

From the root directory execute:

```bash
.\run.bat
```

The Scalar API documentation is available at http://localhost:5006/scalar.
